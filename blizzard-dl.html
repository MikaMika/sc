A Battle.net Downloader for Blizzard Classic Games

Installers

Get a Blizzard Downloader for a specific game. This is a small Torrent client set up to download the Installer and all required files. Installer requires a valid CD Key.

StarCraft will require a 26 character CD-key. On physical media, there is usually a 13 character CD-key. The way to get a 26 was to Redeem a Code. Since Remaster this seem to not work anymore.
